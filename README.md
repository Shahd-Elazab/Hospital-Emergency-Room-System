# 🏥 Hospital Emergency Room Triage Engine

[![C++ Standard](https://img.shields.io/badge/C%2B%2B-11%20%2F%2014%20%2F%2017-blue.svg)](https://en.wikipedia.org/wiki/C%2B%2B)
[![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)](https://microsoft.com)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A high-performance, low-latency triage scheduling and patient tracking simulator engineered for high-pressure clinical environments. Abandoning traditional first-come, first-served array paradigms, this production-grade software dynamically scores and reroutes patient prioritization in real-time by evaluation of acute active complaints overlaid against persistent historical medical profiles. 

Developed as a core data structures framework for **CSE123: Data Structures and Algorithms** at the Egypt University of Informatics (EUI).
